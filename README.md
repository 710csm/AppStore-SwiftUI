# AppStore-SwiftUI

## 프로젝트 정보
- 사용한 개발 언어: Swift
- 사용한 UI 프레임워크: SwiftUI
- 사용한 디자인 패턴: TCA + Clean Architecture
- 사용한 라이브러리
    - CocoaLumberjack: Log
    - SwiftData: Database 
    - Moya: Network 
    - swift-snapthot-testing: SnapShot Test
